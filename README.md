## **ISEL Game Studios Launcher**

This project aims to help ISEL Game Studios keep their games available on a portable platform for use at external events.
The application is designed to be simple and intuitive, allowing even users with little technical experience to use it easily. 
It will be developed using Electron with web technologies, and is intended to be easy to maintain and upgrade over time

### **Main Objectives**

Using the application, users must be able to:

- Add and remove games from the library;

- Save the game library to a JSON file, making it easily transferable;

- Change grid styles and switch between dark and light modes to improve usability.
