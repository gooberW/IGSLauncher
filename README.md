# IGS Launcher

## Features
- Add games to the collection with:
  - Cover image
  - Icon
  - Executable path
  - Tags
  - Description
- Edit and remove games

- Theme customization:
   - Fully customizable colors
   - 5 built-in themes

- Search the game library with autocomplete
- Change view modes (grid and list)
- Library sorting
- Filtering by tags

## Install
To install the app, simply run the setup file and follow the installer instructions.

## Changing Themes
You can customize the app's appearance from the **Settings** page.

- Adjust individual colors manually

- Or choose from one of the 5 pre-made themes

## **Future Versions**
The following features are planned but not yet implemented:
